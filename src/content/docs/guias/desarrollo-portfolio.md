## Introducción

Este documento sirve como prueba para verificar el correcto funcionamiento de **Starlight** dentro de nuestro proyecto base de Astro. 

Aquí documentaremos cómo se construyó este portfolio profesional.

### Arquitectura Híbrida

Hemos integrado dos sistemas que conviven en el mismo repositorio:

1. **Cactus Theme**: Se encarga de la *landing page*, el blog y el listado de proyectos.
2. **Starlight**: Toma el control exclusivo de la ruta de documentación (como esta página), proporcionando un menú lateral autogenerado, tabla de contenidos e interfaz de lectura óptima.

* ✅ Limpiar el contenido de ejemplo del theme.
* ✅ Crear landing page a pantalla completa.
* ✅ Verificar el renderizado de Starlight.
* ⏳ Desplegar en GitHub Pages mediante GitHub Actions.
