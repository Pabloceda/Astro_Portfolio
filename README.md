# Pablo Calderón - Portfolio & Documentación Técnica

Este es el repositorio oficial de mi portfolio profesional, desarrollado con **Astro**, **Tailwind CSS** y **Starlight**. El sitio sirve como una carta de presentación de mi perfil como Técnico en Administración de Sistemas Informáticos en Red (ASIR) y como un centro de documentación técnica para mis proyectos, incluido mi Trabajo de Fin de Grado (TFG).

🌐 **URL del Sitio:** [https://pabloceda.github.io/](https://pabloceda.github.io/)

## 🚀 Tecnologías Principales

- **Astro:** Framework hiper-optimizado y rápido para sitios estáticos.
- **Starlight:** Motor de documentación oficial de Astro para la base de conocimientos y guías técnicas.
- **Tailwind CSS v4:** Para un diseño responsivo, fluido y moderno.
- **Efectos Visuales Premium:**
  - Cursor de seguimiento magnético (Cuberto Mouse Follower).
  - Fondo interactivo con caída de meteoritos y colas reactivas al tema.
  - Efectos tipográficos como *Typewriter* y parpadeos asíncronos.
- **Despliegue Automático (CI/CD):** Configurado mediante GitHub Actions para publicación continua en GitHub Pages.

## 📂 Estructura del Proyecto

- `/src/pages/`: Páginas principales del portfolio (Landing page, Blog, etc).
- `/src/content/docs/`: Documentación y manuales gestionados por Starlight.
- `/src/content/post/`: Entradas del blog técnico.
- `/src/components/`: Componentes reutilizables (cursor, meteoros, iconos).
- `astro.config.ts`: Configuración global del sitio y de Starlight.
- `src/site.config.ts`: Metadatos y personalización general.
